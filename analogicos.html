Bloques Analógicos

Los bloques analógicos permiten el tratamiento de señales continuas dentro del PLC, procesando valores numéricos que representan magnitudes físicas como temperatura, presión o velocidad. Su función principal es facilitar el control y ajuste de procesos en los que los valores varían de manera gradual.

Instrucción Aritmética

Permite combinar operaciones matemáticas como suma, resta, multiplicación y división, con posibilidad de incluir paréntesis. Se emplea para el cálculo entre variables analógicas y admite redondeo al número entero más cercano.

Comparador Aritmético

Compara dos señales analógicas aplicando factores de ganancia (A) y desplazamiento (B). Genera una salida booleana cuando la comparación cumple una condición establecida (>, <, =, etc.).

Conmutador Analógico de Valor Umbral

Activa una salida digital cuando una señal analógica supera o cae por debajo de un valor de referencia (umbral). Es útil para generar alarmas o activar actuadores al alcanzar ciertos límites.

Amplificador Analógico

Multiplica la señal de entrada por una ganancia (A) y suma un desplazamiento (B), lo que permite amplificar o ajustar la escala de una variable analógica.

Vigilancia del Valor Analógico

Monitorea una señal analógica y detecta si el valor está dentro o fuera de un rango definido. Puede emplearse para control de seguridad o detección de fallas.

Conmutador Analógico de Valor Umbral Diferencial

Funciona similar al conmutador de umbral, pero incorpora histéresis: dos valores límite (alto y bajo) para evitar oscilaciones al activarse y desactivarse la salida.

Multiplexor (MUX) Analógico

Selecciona una de varias señales de entrada analógicas para conectarla a una sola salida. Es útil cuando se deben procesar múltiples señales en un solo canal de cálculo o visualización.

Rampa Analógica

Genera un valor analógico que varía progresivamente desde un valor inicial hasta uno final a una velocidad determinada. Se utiliza para suavizar transiciones o generar señales de prueba.

Regulador PI

Ejecuta un control proporcional-integral sobre una señal de error entre un valor de consigna y uno medido. Corrige desviaciones de manera continua en sistemas de lazo cerrado.

Generador PWM Analógico

Convierte una señal analógica en una modulación por ancho de pulso (PWM), adecuada para el control de velocidad o intensidad en motores y actuadores.

Filtro Analógico

Reduce el ruido o las variaciones rápidas en una señal analógica, manteniendo únicamente las variaciones suaves. Se usa para estabilizar mediciones o señales de sensores.

Máximo / Mínimo

Compara varias señales de entrada y entrega el valor más alto o más bajo, según se configure. Permite identificar límites en un conjunto de variables analógicas.
